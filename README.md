# JAMOUS_NL - استيراد الآليات الثقيلة الأوروبية

A production-quality, frontend-only Arabic heavy equipment marketplace website built with React, TypeScript, Vite, and Tailwind CSS.

## 🌟 Features

### Pages
- **Home Page**: Hero section, statistics, why choose us, categories, featured equipment, brands, testimonials, and CTA
- **Equipment Listing**: Full catalog with search, filters, pagination
- **Equipment Details**: Individual equipment pages with image gallery, specs, and contact options
- **Categories**: Browse equipment by category
- **About Us**: Company story, mission, vision, values, timeline, and statistics
- **Contact**: Contact form, contact info, Google Maps, and FAQs
- **Privacy Policy**: Complete privacy policy
- **Terms & Conditions**: Complete terms and conditions

### UI Components
- Navbar (sticky, responsive)
- Footer (comprehensive with links and contact info)
- Equipment Cards (with hover effects)
- Category Cards
- Search Bar
- Filter Sidebar
- Breadcrumb Navigation
- Pagination
- Accordion (for FAQs)
- Testimonial Cards
- Loading Skeletons
- Modal Windows
- And more...

### Features & Functionality
- **Smart Search**: Search equipment by name (Arabic & English)
- **Advanced Filtering**: Filter by category, brand, price, year, condition
- **Responsive Design**: Fully optimized for desktop, laptop, tablet, and mobile
- **RTL Support**: Complete right-to-left support for Arabic
- **Premium Animations**: Fade in, card hover lift, image zoom, button ripple, smooth scroll
- **Premium Images**: High-quality equipment images from Pexels
- **SEO Optimized**: Semantic HTML, meta tags, proper heading structure
- **Performance Optimized**: Lazy loading, minimal JavaScript, optimized images

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 📂 Project Structure

```
src/
├── components/        # Reusable UI components
├── pages/            # Page components
├── data/             # Static data (equipment, categories, etc.)
├── utils/            # Utility functions
├── index.css         # Global styles and animations
├── main.tsx          # Application entry point
└── App.tsx           # Main application component with routing
```
